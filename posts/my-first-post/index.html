如何用 hugo 搭建个人博客

  1. 到github上下载合适的hugo安装包hugo release
  2. hugo version 命令查看是否安装成功
  3. hugo new site <网站名> 新建一个网站。如hugo new site 创建一个名为quikstart的网站

  4. 为网站添加一个主题

    cd quickstart
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
echo 'theme = "ananke"' >> config.toml

  5. hugo new posts/my-first-post.md 新建一篇文章，所有文章放在content/posts目录下,可以编辑其中的文章， 如需要发布文章，记得把draft: true改为draft: false

  6. hugo server -D 预览网站，浏览器访问 http://localhost:1313

  7. 网站配置文件是config.toml，如可配置语言选项为languageCode = “zh-Hans”,网站名字等

  8. 运行hugo命令生成静态网页，放在public目录下

  9. <小写的用户名>.github.io个人页面。在github上创建一个名为<用户名>.github.io的仓库，将本地的public目录 推送到该仓库即可。

    在网站目录中新建一个.gitignore文件，忽略public目录
cd public
git init
git add .
git commit
git remote add <刚刚新建的远程仓库地址>
git push -u origin master
